Pakistan's Premier Rewards Platform Telegram Bot

## Features
- Plan verification with ACTIVE2000
- User registration with Firebase
- Daily ad watching (5 ads × 20 PKR)
- Referral system (500 PKR per referral)
- Withdrawal system (JazzCash/Easypaisa)
- Challenge system (3 referrals to unlock 600 PKR bonus)

## Deployment on Railway

1. Push these files to GitHub
2. Connect to Railway.app
3. Add environment variables:
   - `BOT_TOKEN`: Your Telegram bot token
   - `FIREBASE_CONFIG`: Your Firebase service account JSON
4. Deploy!

## Environment Variables
- `BOT_TOKEN`: Required - Your Telegram bot token
- `FIREBASE_CONFIG`: Required - Firebase service account JSON
